# Picklr

Picklr is a framework for managing cloud infrastructure using Pkl configuration language.

## Overview

Picklr combines the power of Pkl's safe, modular configuration with Go-based providers to manage resources across various cloud providers (currently focusing on AWS).

## Directory Structure

- `cmd/`: Application entry points
- `examples/`: Example Pkl configurations
- `internal/`: Internal application code
- `pkg/`: Public library code and schemas
- `providers/`: Resource provider implementations
